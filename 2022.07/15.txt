form

웹 페이지에서 <form> Tag를 이용하여 입력 Tag 들을 감싸주어야 하며 입력받은  데이터를 서버로 보낼 수 있다.

<form action="입력 데이터를 처리할 페이지 주소" method="입력 데이터를 서버에 전달할 방식"></form>

- action 속성 : 입력 데이터를 처리할 페이지 URL

- method 속성 : 입력 데이터를 처리할 페이지 URL
    - get 방식 : 주소에 데이터를 추가하여 서버에 전달하는 방식이며 중요도가 낮은 데이터를
                전달할 때 주로 사용한다.
    - post 방식 : 데이터를 별도로 첨부하여 전달하는 방식이며 보안성 및 활용성이
                  get 방식보다 뛰어나다.

input 

1. 텍스트 입력
    -type 속성 : 타입을 지정해주는 속성이며 해당 예시에서는 text로 지정해주었다.
    -name 속성 : 이름을 지정해주는 속성이며, 데이터를 처리해야하는 페이지에서 사용한다.
    -placeholder 속성 : 무엇을 입력해야 하는지 안내하는 문구를 넣고 싶을 때 사용. 버튼 후 
                        내용은 보이지 않는다. 

2. 패스워드 입력

3. 라디오 버튼 : 아래 예제는 여러개 옵션 중 하나만 선택할 수 있는 radio type
 - cheked 속성 : 초기에 화면을 열 때 체크되어 표시한다.
 - name 속성 : 이름을 지정해주는 속성이며, 데이터를 처리해야하는 페이지에 사용된다.
            같은 radio type 옵션들은 반드시 같은 name 속성을 가지고 있어야 한다.
 - value 속성 : 값을 명시한다.
